# Session 05 – Naive Bayes

## Objective

To build a Naive Bayes classification model and classify Iris flower species using probability-based prediction.

## Dataset Used

iris.csv

## Features

Independent Variables (X):

* sepal_length
* sepal_width

Dependent Variable (y):

* species

## Operations Performed

* Imported dataset using Pandas
* Removed missing values
* Converted categorical values into numeric values
* Split dataset into training and testing sets
* Standardized feature values
* Trained Gaussian Naive Bayes model
* Generated predictions
* Calculated model accuracy
* Generated Confusion Matrix
* Visualized Decision Boundary

## Model Outputs

* Accuracy Score
* Confusion Matrix
* Decision Boundary Visualization

## Real-world Applications

Learned how Naive Bayes performs classification using probability principles and how feature independence affects prediction performance.
