# Session 04 – Logistic Regression

## Objective

To build a Logistic Regression model and classify breast cancer cases using probability-based prediction.

## Dataset Used

breast_cancer.csv

## Features

Independent Variable (X):

* radius_mean

Dependent Variable (y):

* diagnosis

  * 0 → Benign
  * 1 → Malignant

## Operations Performed

* Imported dataset using Pandas
* Removed unnecessary columns
* Converted categorical values into numeric values
* Removed missing values
* Trained Logistic Regression model
* Generated probability predictions
* Visualized Sigmoid Curve
* Calculated coefficient and intercept
* Evaluated model accuracy

## Model Outputs

* Sigmoid Curve
* Coefficient
* Intercept
* Accuracy Score
* Sample Predictions
