# Session 07 – Decision Tree

## Objective

To build a Decision Tree classification model and predict passenger survival using the Titanic dataset.

## Dataset Used

titanic.csv

## Features

Independent Variables (X):

* Pclass
* Age
* Sex
* Fare

Dependent Variable (y):

* Survived

## Operations Performed

* Imported dataset using Pandas
* Selected required features
* Converted categorical values into numeric values
* Removed missing values
* Split dataset into training and testing sets
* Trained Decision Tree Classifier
* Generated predictions
* Calculated model accuracy
* Generated Classification Report
* Created Confusion Matrix
* Visualized Decision Tree

## Model Outputs

* Accuracy Score
* Classification Report
* Confusion Matrix
* Decision Tree Visualization

## Real-world Applications

* Medical diagnosis
* Customer churn prediction
* Loan approval systems
* Fraud detection
* Risk assessment

## Output Files

* confusion_matrix.png

* console_output.png

Learned how Decision Trees classify data using rule-based splitting and how feature selection affects prediction performance.
