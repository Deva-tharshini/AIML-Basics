# Session 08 – Random Forest

## Objective

To build a Random Forest classification model and predict passenger survival using the Titanic dataset.

## Dataset Used

titanic.csv

## Features

Independent Variables (X):

* Pclass
* Age
* Sex
* Fare

Dependent Variable (y):

* Survived

## Operations Performed

* Imported dataset using Pandas
* Selected required features
* Converted categorical values into numeric values
* Removed missing values
* Split dataset into training and testing sets
* Trained Random Forest Classifier
* Generated predictions
* Calculated model accuracy
* Generated Confusion Matrix
* Generated Classification Report
* Visualized Confusion Matrix using Heatmap

## Model Outputs

* Accuracy Score
* Confusion Matrix
* Classification Report
* Heatmap Visualization

## Real-world Applications

* Customer behavior prediction
* Fraud detection
* Medical diagnosis
* Risk assessment
* Recommendation systems

## Output Files

* confusion_matrix.png
* console_output.png

## Learning Outcome

Learned how Random Forest combines multiple decision trees to improve prediction performance, reduce overfitting, and increase model reliability.
