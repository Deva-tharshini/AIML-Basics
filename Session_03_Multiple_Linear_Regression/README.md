# Session 03 – Multiple Linear Regression

## Objective

To build a Multiple Linear Regression model and predict profit using multiple business-related factors.

## Dataset Used

startups.csv

## Features

Independent Variables (X):

* R&D Spend
* Administration
* Marketing Spend

Dependent Variable (y):

* Profit

## Operations Performed

* Imported dataset using Pandas
* Removed missing values
* Selected multiple input features
* Trained Multiple Linear Regression model
* Generated predictions
* Visualized regression relationship
* Calculated coefficients and intercept
* Evaluated model performance using R² Score

## Model Outputs

* Regression Projection Graph
* Coefficients
* Intercept
* R² Score
* Sample Predictions

## Real-world Applications

* Startup profit prediction
* Business analytics
* Investment planning
* Revenue forecasting
* Financial decision support

Learned how multiple independent variables influence prediction outcomes and how regression models identify relationships between features.
