# Session 02 – Linear Regression

## Objective

To build a Linear Regression model and understand the relationship between years of experience and salary prediction.

## Dataset Used

salary.csv

## Features

* YearsExperience → Independent Variable (X)
* Salary → Dependent Variable (y)

## Operations Performed

* Imported dataset using Pandas
* Removed missing values
* Selected features and target
* Trained Linear Regression model
* Predicted output values
* Generated regression visualization
* Calculated coefficient and intercept
* Evaluated model performance

## Model Outputs

* Best Fit Regression Line
* Coefficient (Slope)
* Intercept
* R² Score
* Sample Predictions

## Real-world Applications

* Salary estimation
* Sales forecasting
* Revenue prediction
* Demand forecasting
* Financial trend analysis

## Output Files

* best_fit_line.png
* console_output.png

## Learning Outcome

Learned how Linear Regression models establish relationships between variables and generate predictions using mathematical best-fit lines.
