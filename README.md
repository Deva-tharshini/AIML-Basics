## Project Overview

This repository contains practical implementations of core Artificial Intelligence and Machine Learning concepts completed through structured laboratory sessions.

Each session focuses on one algorithm or AI concept and includes implementation, outputs, documentation, and visual analysis to improve practical understanding.

The repository is organized session-wise and uses a shared dataset folder for better project management and reusability.

## Sessions Included

### Session 01 – Linear Regression

Implemented simple linear regression for numerical prediction.

### Session 02 – Multiple Linear Regression

Predicted output values using multiple independent variables.

### Session 03 – Logistic Regression

Performed binary classification using logistic regression.

### Session 04 – Naive Bayes

Implemented probabilistic classification using Bayes theorem.

### Session 05 – Support Vector Machine (SVM)

Applied kernel-based classification and model comparison.

### Session 06 – Decision Tree

Built interpretable classification models using decision trees.

### Session 07 – Random Forest

Applied ensemble learning using multiple decision trees.

### Session 08 – Ensemble Learning

Implemented Bagging and Boosting algorithms.

### Session 09 – K-Means Clustering

Performed unsupervised clustering and customer segmentation.

### Session 10 – Bayesian Networks

Implemented probabilistic reasoning using Bayesian Network concepts.

### Session 11 – Expectation Maximization (EM)

Applied Gaussian Mixture Models for clustering and segmentation.

### Session 12 – Bayesian Networks

Built Bayesian dependency structures and performed probabilistic inference.

### Session 13 – Neural Networks

Implemented:

• Single Layer Neural Network (SLNN)

• Multi Layer Neural Network (MLNN)

### Session 14 – Search Algorithms

Implemented:

• Breadth First Search (BFS)

• Depth First Search (DFS)

• A* Search Algorithm
